# Directus на Railway

### Кроки для деплою
1. Додай цей репо на GitHub.
2. На Railway натисни **New Project → Deploy from GitHub**.
3. Додай плагін PostgreSQL на Railway.
4. Встанови Environment Variables:
    - DB_HOST
    - DB_PORT
    - DB_DATABASE
    - DB_USER
    - DB_PASSWORD
5. Деплой — Railway сам підхопить Docker.
6. Зайди на URL від Railway → увійди в адмінку.
